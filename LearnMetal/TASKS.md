# Metal Shaders - Задания

## Уровень 0 — Фундаментальные основы

### 0.1 Флаг ✅
Нарисуй трёхполосный флаг (например Франции) — три вертикальные цветные полосы.
Понадобятся: `position`, `size`, условия.
**Изучено**: `step()`, `mix()`, UV-координаты, нормализация.

### 0.2 Шахматная доска ✅
Нарисуй шахматную доску 8x8 из чёрных и белых клеток.
Подумай, как из нормализованной координаты определить чётность клетки.
**Изучено**: `floor()`, `fmod()`, разница между `floor(a+b)` и `floor(a)+floor(b)`.

### 0.3 Инверсия ✅
Левая половина экрана — градиент (тёмный -> светлый), правая — инвертированный (светлый -> тёмный). Переход резкий.
**Изучено**: инверсия через `1.0 - x`, UV как яркость напрямую.

### 0.4 Круговой градиент ✅
Градиент от центра экрана к краям — белый в центре, чёрный по краям.
Без `step`, без `smoothstep` — только базовая математика и нормализация.
**Изучено**: `length()`, расстояние до точки, нормализация dist, `clamp()`.

### 0.5 Виньетка ✅
Цветная виньетка — края затемняются, центр яркий.
R зависит от расстояния до центра, G и B — от координат.
**Изучено**: независимое управление RGB-каналами, затухание через умножение.

### 0.6 Микс каналов ✅
- R = горизонтальный градиент
- G = вертикальный градиент
- B = R * G
**Изучено**: произведение каналов, предсказание цвета по формуле.

### 0.7 Полосы ✅
Чередование двух цветов, 10 полос на экран.
**Изучено**: 1D-паттерн через `floor()` + `fmod()`, кастомные цвета через `half4(r/255, g/255, b/255, 1)`.

### 1.2 Прямоугольник [В ПРОЦЕССЕ — теория]
Нарисуй прямоугольник по центру. Размер передаётся из Swift в поинтах.

**Где остановились:**
Разобрали формулу `d = abs(centered) - half_size` — расстояние до края по каждой оси.
Разобрали три случая пикселя: внутри (оба d < 0), сбоку (один d > 0), за углом (оба d > 0).
Поняли зачем `max(d, 0)` — обнулить оси где пиксель внутри, чтобы не искажать length().
**Осталось:** как посчитать расстояние для пикселя ВНУТРИ прямоугольника + собрать итоговую формулу SDF.
